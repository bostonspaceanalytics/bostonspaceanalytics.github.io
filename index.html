Boston Space Analytics

Boston Space Analytics (BSA) is focused on characterizing the estimated 10,000 known, but uncharacterized, large near earth asteroids (NEAs). Various groups of amateur and professional astronomers have discovered a total of 11,460 NEAs (registered). However, only 10% are currently characterized. No astronomical team has undertaken the task to systematically characterize all the discovered NEAs. BSA aims to fill this gap.

More

The Goal

By Congressional mandate, NASA has been focused on discovering potentially hazardous asteroids. But very little effort and funding has been devoted to characterization studies. The 2020 Asteroid Redirect Mission (ARM) is aiming at bringing back an 8m NEA. Selecting the best NEA from the estimated 100 million NEAs of this size is critical. In addition to orbital information, it is necessary to consider other characterizations such as composition, size, density, mass, type, spin, rotation and shape.

Characterization

Asteroid mining firms and governmental agencies, like NASA, need commercial “mining tool” companies to provide data analytics similar to seismic studies used for Earth mining. BSA is a for-profit startup that will focus only on providing NEA characteristics. BSA’s team has deep expertise in visual imaging analytics and astronomy. We plan to deliver NEA characteristics using autonomous processes.

Images

BSA will use leased time on several 1m and 2m telescopes in 5 locations distributed around the world. The global locations ensure that weather or technical issues do not impede data gathering. We will build a pipeline to autonomously select the NEAs. For each NEA, we will use optical and near-IR imaging, photometric light curves, optical color, and spectroscopy as appropriate. We aim to use these astronomy methods to systematically characterize all known observable, but as yet uninvestigated, NEAs.

Analytics

Images will be uploaded in real-time to a cloud database. Quality assurance methods and noise reduction algorithms will be applied. Visual artificial intelligent coding embedded with proper machine learning methods to handle Big Data will extract patterns from the data. The visual analytic code will be similar to facial recognition code. The coding will deliver characteristics of the NEAs autonomously. The results will be monitored by machine learning code against academically agreed upon NEA characteristics.
